# Exercicios de Python

Lista de exercicios para praticar aos poucos, acompanhando a trilha do projeto.

## Como usar

- Resolva os exercicios na ordem.
- Crie um arquivo novo quando quiser praticar.
- Tente fazer sozinho antes de olhar exemplos anteriores.
- Se travar, volte um nivel e revise.

## Nivel 1: variaveis e operadores

1. Crie variaveis para nome, idade e cidade, depois mostre tudo na tela.
2. Guarde dois numeros e mostre soma, subtracao, multiplicacao e divisao.
3. Calcule a idade da pessoa em dias.
4. Converta metros para centimetros.
5. Calcule a area de um quadrado.

## Nivel 2: condicoes

1. Verifique se uma pessoa e maior de idade.
2. Verifique se um numero e positivo, negativo ou zero.
3. Descubra se um numero e par ou impar.
4. Receba uma media e informe se o aluno foi aprovado, recuperacao ou reprovado.
5. Compare dois numeros e diga qual e maior.

## Nivel 3: loops

1. Mostre os numeros de 1 a 10 usando `for`.
2. Mostre os numeros de 10 a 1 usando `while`.
3. Crie a tabuada de um numero.
4. Some todos os numeros de 1 ate 100.
5. Mostre apenas os numeros pares entre 1 e 20.

## Nivel 4: funcoes

1. Crie uma funcao que receba um nome e exiba uma saudacao.
2. Crie uma funcao que some dois numeros e retorne o resultado.
3. Crie uma funcao para calcular o IMC.
4. Crie uma funcao que verifique se o numero e par.
5. Crie uma funcao que receba tres notas e devolva a media.

## Nivel 5: listas e dicionarios

1. Crie uma lista com cinco frutas e mostre uma por linha.
2. Adicione um item novo a uma lista e mostre o resultado.
3. Some todos os numeros de uma lista.
4. Crie um dicionario com nome, idade e profissao.
5. Mostre apenas as chaves de um dicionario.

## Nivel 6: arquivos

1. Crie um arquivo `.txt` e escreva uma frase nele.
2. Leia o conteudo do arquivo e mostre na tela.
3. Salve tres nomes em um arquivo.
4. Leia o arquivo de nomes e mostre cada linha separadamente.
5. Crie um mini bloco de notas simples.

## Nivel 7: tratamento de erros

1. Tente dividir um numero por zero e trate o erro.
2. Tente converter um texto para numero e trate o erro.
3. Mostre uma mensagem amigavel quando o usuario digitar algo invalido.
4. Leia um arquivo que nao existe e trate o erro.
5. Use `finally` para mostrar uma mensagem de encerramento.

## Nivel 8: classes e objetos

1. Crie uma classe `Pessoa` com nome e idade.
2. Crie uma classe `Aluno` com nome e nota.
3. Adicione um metodo para verificar se o aluno foi aprovado.
4. Crie uma classe `Produto` com nome e preco.
5. Crie uma classe `ContaBancaria` com saldo e metodo para depositar.

## Mini projetos

1. Calculadora simples no terminal.
2. Sistema de notas de aluno.
3. Lista de tarefas.
4. Agenda de contatos.
5. Cadastro de produtos.

## Quando avancar

Voce pode subir de nivel quando:

- entende o que o codigo esta fazendo sem decorar
- consegue mudar os valores sozinho
- consegue criar uma versao parecida do zero
- consegue explicar com suas palavras
